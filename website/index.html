Deploy Your Agent

Launch autonomous agents on WhatsApp, Telegram, Twitter, or Farcaster in minutes. 

# Quick start
npm install
cp .env.example .env

# Configure platforms in .env:
# WhatsApp (via Twilio):
#   TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_WHATSAPP_FROM
# Telegram:
#   TELEGRAM_BOT_TOKEN
# Twitter:
#   TWITTER_API_KEY, TWITTER_API_SECRET
#   TWITTER_ACCESS_TOKEN, TWITTER_ACCESS_SECRET
# AI Models:
#   ANTHROPIC_API_KEY (Claude), OPENAI_API_KEY (GPT-4)

# Start agent
npm run sentient
