# Warelay Agent

**Framework for building autonomous AI agents on WhatsApp, Telegram, Twitter, and Farcaster.**

Warelay is an open-source infrastructure for deploying conversational AI agents that remember context, adapt personality, and operate independently across multiple messaging platforms.

## Quick Start

```bash
# Install dependencies
npm install

# Configure your credentials
cp .env.example .env

# Start your agent
npm run sentient
```
